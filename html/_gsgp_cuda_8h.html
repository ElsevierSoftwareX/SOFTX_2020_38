Detailed Description

File containing the definition of the modules (kernels) used to create the population of individuals, evaluate them, the search operator and read data.

Structure that represents the tuple used to store information on the individuals involved in each generation and is used for the reconstruction of the best individual.

Structure used to store the parameters of the configuration.ini file and these are used to initialize the algorithm parameters

created on 25/01/2020
Parameters
intnumberGenerations: number of generations of the GSGP algorithm
intpopulationSize: number of individuals in the population
intmaxIndividualLength: variable that stores the length (number of genes) of an individual

floatvariableRatio : probability of selecting a variable (otherwise a constant) when a terminal gene has been chosen.
